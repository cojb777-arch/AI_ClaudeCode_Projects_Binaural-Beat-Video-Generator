# 🎧 Binaural Beat Video Maker

バイノーラルビート × サブリミナル音声 × サブリミナルフラッシュ画像を組み合わせた動画を、ブラウザだけで生成できるアプリです。依存ライブラリなし・単一HTMLファイルで動作します。

## 使い方

`index.html` をブラウザ（Chrome / Edge 推奨）で開くだけです。

1. 🎧 ステレオヘッドホンを装着する
2. うねり周波数（1〜30Hz）を選ぶ — プリセット: δデルタ(睡眠) / θシータ(瞑想) / αアルファ(リラックス) / βベータ(集中)
3. 任意でサブリミナル音声（自分のアファメーション録音など）をアップロード → 微小音量でループ再生
4. 任意でフラッシュ画像・メッセージ文を設定（17〜100msの瞬間表示）
5. 「▶ プレビュー」で確認 → 「● 録画開始」で WebM 動画を生成・自動ダウンロード

## 仕組み

- **バイノーラルビート**: Web Audio API で左耳に基本周波数、右耳に「基本＋うねり」の正弦波を再生し、脳内でうねり（周波数差）を知覚させます
- **動画生成**: Canvas 映像と音声を MediaRecorder でリアルタイム録画（5分の動画＝5分の録画時間）。出力は WebM 形式（YouTube にそのままアップロード可）

## 安全への配慮

- フラッシュ間隔は最短3秒・表示は最長100msに制限（光過敏性発作対策）
- うねり3Hz超では画面の発光振幅を自動的に抑制
- 録画時はサブリミナル音声の試聴用大音量を強制OFF

⚠️ てんかん・光過敏性発作の既往がある方、心臓疾患・妊娠中の方は使用を控えてください。機器の運転中は使用しないでください。
